My Set Up

Laptop

The laptop I currently own it's the Macbook Air M2. Here are the specifications:

  • 8-core CPU with 4 performance cores and 4 efficiency cores
  • 16-core Neural Engine
  • 8GB of RAM
  • 256GB SSD
Monitor

The monitor I'm using it the SAMSUNG 27" M50B Series FHD Smart Monitor. It's a very good monitor because it's also a TV, so you can get the best of monitor and TV.

  • Screen size: 27"
  • Model: LS27BM501ENXZA
  • Refresh Rate: 60 Hz
  • Response Time: 4ms
  • Display Type: Full HD
Mouse

The mouse I'm currently using and that I'm in love with (it's really good!) it the Alienware Wired/Wireless Gaming Mouse AW610M. Mine is the "Lunar Light" color, but you can also get it in black which is the "Dask Side of The Moon".

  • Movement Resolution: 16000 dpi
  • Buttons Qty: 7
  • Interface: USB
Keyboard

I recently purchased the Apple Magic Keyboard with numeric keypad. I'm really enjoying it, because it connects with Macbook by Bluetooth, so you don't need any adaptor to make it work on your laptop, it connects automatically.
